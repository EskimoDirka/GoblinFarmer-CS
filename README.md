# GoblinFarmer-CS

A WinForms C# Diablo III farming assistant.

## Features

- Teleport routing system
- Teleport Next hotkey
- Location verification
- Teleport blocking logic
- Combat automation
    - Monk
    - Demon Hunter
    - Witch Doctor
- Make New Game flow
- Leave Game flow
- Repair automation
- Salvage automation
- Image recognition with scan region support
