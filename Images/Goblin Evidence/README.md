# Goblin Evidence Templates

Place calibrated observation templates in this folder:

- `Journal Kill.png`
- `Journal Encounter.png`
- `Minimap Goblin Icon.png`

Use Ctrl+Shift+G calibration captures or `Debug\GoblinEvidence\ObservationDiagnostics` crops as reference material only. Do not auto-create these templates from arbitrary crops; manually crop and validate them before enabling observation candidate detection.
