# Gem Stash Assets

Put gem stash PNG templates directly in this folder. Any `*.png` whose name does not include helper words such as `stash`, `tab`, `scan region`, or `placement` is treated as a stash-eligible gem template.

Create `Gem Coordinates.txt` with the existing named-coordinate format:

```text
========== Stash Coordinates ==========
287, 471

========== Gem Stash Tab Coordinates ==========
680, 701
```
